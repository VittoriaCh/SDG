## 二、NPP 原始数据质量问题

### 2.1 2000 年数据严重偏低（最核心的数据质量问题）

**发现**：2000 年的 NPP Proxy 均值仅为 **8.35**，而后续所有年份的均值在 **13.46–14.45** 之间。2000 年比后续年份低约 **38%**。

| 年份 | 均值 | 中位数 | 与 2000 年的差距 |
|------|------|--------|----------------|
| 2000 | **8.35** | 8.28 | — |
| 2001 | 13.69 | 13.48 | +64% |
| 2002 | 13.58 | 13.41 | +63% |
| ... | ... | ... | ... |
| 2023 | **14.45** | 14.06 | +73% |

在 125 万个全时段有效像素中：
- 89.0% 的像素在 2000 年的值 < 10
- 仅 9.7% 的像素在 2001 年的值 < 10
- **99.7% 的像素从 2000 到 2001 年都出现了上升**，平均增加 5.35

**可能原因**：
1. **MODIS Terra 传感器早期校准问题**：2000 年是 Terra 卫星发射后的第一个完整年份，MOD13Q1 Collection 6.1 对早期数据的 NDVI 反演可能存在系统性偏差
2. **生长季窗口定义对 2000 年不适用**：生长季窗口（peak_doy ± 64 天）是基于多年平均曲线计算的，如果 2000 年的物候特征与多年平均差异大，窗口内的 NDVI 积分会显著偏低
3. **MODIS 数据缺失导致的插值偏差**：2000 年的 NaN 像素略多于后续年份（2,079,494 vs 平均 2,078,000），缺失模式不同可能影响窗口内积分

**对分析结果的影响**：

**Trend 指标**：2015 年报告期（窗口 2000–2015）是唯一包含 2000 年的窗口。所有 (2000, Y) 的点对斜率都为正（因为 2000 年极低），在 120 个点对中占 15 个。这直接导致：
- 2015 年 Theil-Sen 斜率中位数为 **+0.022**（所有报告期中最正）
- 改善面积 2,891 km²（远高于 2016–2022 年的 1,500–2,000 km²）
- 退化面积 1,329 km²（远低于 2016–2022 年的 2,200–2,900 km²）

**2015 年的低退化不是真实的生态改善，而是 2000 年数据异常造成的统计偏差。**

**State 指标**：2015 年报告期的 H0（2000–2012）包含 2000 年，H1（2013–2015）不包含。2000 年的极低值拉低了 H0 均值，导致 H1 - H0 偏正，更多的像素被判为改善。这解释了为什么 2015 年的 State 退化面积仅 420 km²（所有年份最低）。

**Performance 指标**：Performance 使用的是 16 年平均值，包含 2000 年的窗口平均值会被拉低约 (13.6 - 8.35) / 16 ≈ 0.33。这意味着包含 2000 年的窗口中，mean_npp 偏低，ratio 偏低，退化面积应偏大。但 Performance 的退化面积在 2015 年是所有年份中最低的（2,446 km²），说明 2000 年对 Performance 的影响不如对 Trend/State 显著（因为 P90 基准也会随窗口变化而调整）。

### 2.2 负 NPP 值（物理上不可能）

NPP Proxy 是生长季 NDVI 曲线的离散积分（AUC），理论上应 ≥ 0。但数据中每一年都存在负值：

| 统计 | 数值 |
|------|------|
| 每年代负值像元数 | 2,811–3,911 |
| 最小值 | -18.98（2022 年） |
| 负值均值 | 约 -2.7 |
| 在所有 26 年中均为负的像元 | 仅 113 个 |

负值像元仅占有效像素的 0.2%，对区域统计影响有限，但在像素级分析中应予以掩膜。

### 2.3 大量 NaN 像素

- 62.2% 的像素在全部 26 年中均为 NaN（沙漠无植被区域）
- 仅 37.5% 的像素在所有 26 年中都有有效值
- 0.2% 的像素在部分年份有 NaN（8,112 个像素）

这意味着分析仅基于约 125 万个像素，且这些像素主要分布在有植被覆盖的区域（城市边缘、绿洲、沿海地带），不代表 UAE 全境。

### 2.4 2023 年异常高值

2023 年的 NPP 均值为 **14.45**，是 26 年中最高的，比序列均值（~13.7）高出 5.5%。这可能是真实的（如 2023 年降水充沛），但也需要警惕数据质量问题。

## 三、异常现象逐条分析

### 3.1 Trend 指标：2000 年异常导致 2015 年虚假改善

#### 3.1.1 2015 年退化面积异常低（1,329 km²）

**根本原因**：2000 年数据偏低 38%。

2015 年窗口（2000–2015）的 Theil-Sen 斜率中位数为 +0.022，是所有报告期中最正的。120 个点对中，所有涉及 2000 年的 15 个点对都为正斜率（平均斜率 +0.33），将中位数向右拉动。

当 2000 年移出窗口后（2016 年报告期，窗口 2001–2016），斜率中位数降至 +0.006，退化面积从 1,329 km² 飙升至 2,505 km²。

#### 3.1.2 2019 年退化峰值（2,876 km²）

2019 年窗口（2004–2019）的中位数斜率为 **-0.006**，是所有报告期中最负的。这是因为：
- 窗口不包含 2000 年（正偏差来源被移除）
- 窗口起始年 2004 年的均值（13.69）高于窗口中段和末段
- 2015 年（13.50）和 2019 年（13.61）都是序列中的低值年

负中位数斜率导致更多像素的 MK Z 值 < -1.96。

#### 3.1.3 2023–2025 年急剧改善

2023 年窗口（2008–2023）起斜率中位数转正（-0.002），到 2025 年窗口（2010–2025）中位数斜率为 +0.004。

驱动力来自窗口末端的高 NPP 年份：
- 2023 年（14.45）是序列最高值
- 2024 年（14.28）和 2025 年（14.24）也高于序列均值
- 这些高值年与窗口早期年份形成的点对产生大量正斜率

但需警惕：2023 年本身的 NPP 数据是否可靠？如果是数据质量问题（如 MODIS 传感器后期漂移、大气校正算法更新），则 Trend 近期改善是虚假的。

### 3.2 State 指标：3 年评估期 + 2000 年异常 = 极端波动

#### 3.2.1 2015 年异常低值（退化 420 km²）

2015 年 H0 = [2000–2012]，H1 = [2013–2015]。

2000 年的极低值（8.35）将 H0 均值拉低。计算：如果 H0 中剔除 2000 年，H0 均值从 13.33 升至约 13.60。H1 - H0 的差值从 +0.31 降至接近 0。

**2015 年的 State 低退化完全是 2000 年数据异常的产物。**

#### 3.2.2 2016 年退化暴增（5,863 km²）

2016 年 H0 = [2001–2013]，H1 = [2014–2016]。

- H0 不再包含 2000 年，均值升至 13.74
- H1 包含 2015 年（13.50，低值年），H1 均值仅 13.63
- H1 - H0 = -0.11，Z 中位数 = -0.62
- 退化像元 231,286（所有年份中最多之一）

从 2015 到 2016，仅因为窗口滑动移除了 2000 年并加入 2016 年，退化面积就增加了 **14 倍**。这种幅度的变化不可能反映真实的生态变化。

#### 3.2.3 2020 年第二个退化峰值（5,160 km²）

2020 年 H0 = [2007–2019]，H1 = [2018–2020]。

- H0 包含 2007 年（14.07，当时的高值年）
- H1 包含 2018（13.63）和 2019（13.61），都是低值年
- H1 - H0 = -0.02，但 Z 中位数为 -0.69（std_H0 较小，放大 Z 值）
- 退化像元 218,628

#### 3.2.4 2023–2024 年改善暴涨

2023 年 H1 = [2021–2023]，包含 2023 年（14.45，最高值）。
2024 年 H1 = [2022–2024]，包含 2023（14.45）和 2024（14.28）。

H1 - H0 差值从 2022 年的 +0.15 扩大到 2024 年的 +0.47。改善像元从 167,838（2022）飙升至 306,132（2024）。

**但 12,334 km² 的改善面积占 UAE 总面积的 ~15%，在 3 年内完成如此大规模的改善在生态上不可信。** 2023 年的 NPP 高值本身可能需要独立验证。

#### 3.2.5 State 指标的总体评价

State 在 10 个报告期中，退化面积从 420 到 5,863 km²（相差 14 倍），改善面积从 794 到 12,334 km²（相差 15.5 倍）。这种幅度的波动远超出任何合理的生态变化范围，说明：

1. **3 年评估期对单一年份极端值过度敏感**
2. **2000 年的数据异常对 H0 基准造成系统性偏差**
3. **State 结果不应被解读为年度退化/改善趋势**

### 3.3 Performance 指标：方法学驱动的单调趋势

Performance 的退化面积完美单调递增（2,446→2,913 km²，每年增加 47–87 km²）。

这是窗口滑动下 P90 基准漂移的必然结果：
- 窗口从 [2000–2015] 移至 [2009–2024]
- 像元 mean_npp 的下降速度 > P90 的下降速度（P90 是区域统计量，变化缓慢）
- 比率 mean_npp / P90 逐年走低
- 越来越多像元从 bin 2（中等）落入 bin 1（退化）

Performance 的单调性在某种程度上可以视为 "独立于 Trend/State 的系统性信号"——它表明在 UAE 沙漠环境下，即使 Trend 显示改善，像元的绝对生产力水平相对于生态区基准仍在缓慢下降。

### 3.4 组合结果：被 Performance 主导

Table 4-5 规则下，Class 7（N-N-Y，仅 Performance 为退化）判定为退化。由于 Performance 的退化面积最大（2,446–2,913 km²）且最稳定，组合结果中大部分退化像素来自 Performance 的"捕获"。

改善面积极少（44–132 km²），因为：
1. Performance 的改善阈值 ratio ≥ 1.5 极高，在 UAE 沙漠环境下仅极少数像元能达到
2. 即使 Trend 和 State 显示改善，如果 Performance 非改善，组合结果中的改善像素仍然很少

2024 年改善面积增至 132 km²，恰好对应 State 改善面积的极端峰值（12,334 km²）。State 的暴涨释放了更多 Trend=非退化 + State=改善 + Performance=非退化的像素组合。

## 四、总结

### 4.1 数据质量问题清单

| 问题 | 严重程度 | 受影响指标 | 影响描述 |
|------|---------|-----------|---------|
| 2000 年数据偏低 38% | **严重** | Trend, State | 2015 年报告期的低退化是虚假的；所有含 2000 年的窗口结果不可信 |
| 负 NPP 值 | 轻微 | 全部 | 每年代 3,000 个左右负值像素，应掩膜 |
| NaN 像素占 62.5% | 中等 | 全部 | 分析仅基于有植被覆盖区域，不代表 UAE 全境 |
| 2023 年 NPP 异常高 | 需验证 | Trend, State | 如果 2023 年数据不可靠，近期改善趋势被夸大 |
| 3 年评估期 | 方法学 | State | 年度间波动幅度 14–15 倍，超出合理生态变化范围 |

### 4.2 各指标结果可信度评估

| 指标 | 可信度 | 主要问题 |
|------|--------|---------|
| Trend（不含 2015） | 中 | 2019–2022 平台期可信；2023–2025 需验证 2023 年数据 |
| Trend（2015） | **低** | 2000 年异常导致虚假改善信号 |
| State | **低** | 3 年评估期 + 2000 年异常 = 结果不可作为趋势解读 |
| Performance | 中 | 单调趋势为方法学伪影，但方向性信号可能有参考价值 |
| Combined | 中低 | 被 Performance 主导，改善面积代表性不足 |

### 4.3 建议

1. **剔除 2000 年数据**：将起始年改为 2001，重新运行全部评估。这将消除 2000 年异常对所有报告期的影响。
2. **延长 State 评估期**：从 3 年改为 5 年（H0: 11 年，H1: 5 年），降低单一年份极端值的影响。
3. **调整 Performance 分箱边界**：将改善阈值从 1.5 降至 1.0 或 1.2。
4. **掩膜负 NPP 值**：在数据预处理中将负值设为 NaN。
5. **独立验证 2023 年 NPP**：对比 Sentinel-2 或 Landsat NDVI，确认 2023 年高值是否真实。
6. **报告不确定性范围**：由于 State 波动极大，建议在报告中报告多期运行的结果范围，而非单一数值。
